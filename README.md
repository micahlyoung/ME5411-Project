
## ME5411 Project

#### Files:

| File | Description |
| --- | --- |
| `ME5411_CA_2025.pdf` | project requirement |
| `preprocess.m` | image preprocessing |
| `cnn.m` | train CNN |
| `mlp.m` | train MLP |
| `vit.m` | train ViT |
| `train_val_visualization.m` | plot training and validation metrics |

#### Outputs:

| Output | Description |
| --- | --- |
| `outputs/` | generated images and trained models |
| `outputs/preprocess_output/` | intermediate and processed images |
| `outputs/segmentation_output/` | cropped character images |
| `outputs/cnn_output/` | CNN model and training plots |
| `outputs/non_cnn_output/` | MLP and ViT model files |

#### Quick run:
- Run `preprocess.m` then `cnn.m`, `mlp.m`, `vit.m` to reproduce results.
